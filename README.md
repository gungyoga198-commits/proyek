
## 🧪 TECH STACK

```bash
Backend   : Laravel (PHP Framework)
Frontend  : Blade / Bootstrap

Server    : Localhost / Laragon / XAMPP
```

---

## 🧬 STRUKTUR SYSTEM

```bash
/app            -> Logic utama (Controller, Model)
/routes         -> Web routes
/resources      -> Tampilan UI (Blade)
/database       -> Migration & Seeder
/public         -> Asset (CSS, JS, Image)
```

---

## ☣️ INSTALLATION (DEPLOY OR DIE)

```bash
git clone https://github.com/gungyoga198-commits/proyek.git

cd proyek

composer install

cp .env.example .env

php artisan key:generate

# setup database di .env dulu
php artisan migrate

php artisan serve
```
